## 🔧 이미지 수정 요청 예시

만약 첫 결과가 마음에 안 들면:

### 너무 복잡할 때:
```
Make it more minimalist. Remove facial details. 
Use simple silhouette with clean lines only.
```

### 제스처가 불명확할 때:
```
Make the "shh" gesture more prominent and clear.
The finger should be clearly touching the lips.
```

### 색상 조정:
```
Use a darker blue-purple gradient background.
Make the silhouette brighter for better contrast.
```

### 크기/위치 조정:
```
Center the silhouette better.
Make the figure slightly larger to fill more space.
```

## 📐 기술 사양

### 필수 크기
- **원본:** 1024 x 1024 픽셀
- **형식:** PNG (투명 배경 아님, 배경색 있어야 함)
- **색상:** RGB

### 자동 생성할 크기들
appicon.co나 다른 도구로 자동 생성:
- 1024x1024 (App Store)
- 180x180 (iPhone @3x)
- 120x120 (iPhone @2x)
- 87x87 (Settings @3x)
- 80x80 (Spotlight @2x)
- 60x60 (Notification @3x)
- 58x58 (Settings @2x)
- 40x40 (Notification @2x)
